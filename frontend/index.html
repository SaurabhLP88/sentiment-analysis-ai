NLP - Sentiment Analysis using Huggingface

AI-powered Sentiment Analysis web application built using Flask that analyzes user-provided text and classifies it as positive, negative, or neutral along with a confidence score. The application integrates with modern NLP models via external APIs and includes robust error handling with fallback logic to ensure consistent performance even when external services are unavailable. It features a simple and responsive user interface where users can input text and instantly view results, demonstrating practical implementation of backend APIs, asynchronous request handling, and real-world debugging of third-party integrations. This project highlights full-stack development skills, including API integration, testing with mocks, and deployment-ready architecture.
